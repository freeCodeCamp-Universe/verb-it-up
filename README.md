# Verb It Up - README

**Verb It Up** is a browser-based matching game for practicing English phrasal verbs and their definitions.

## Key Features

**Core Gameplay**: Players match phrasal verbs to their definitions by dragging a verb card onto its definition, or by clicking a verb then clicking a definition. Each round presents 5 randomly selected verbs.

**Timer Modes**: Players can choose from four time settings — no timer, 30 s, 60 s, or 90 s — adding a speed challenge to each round.

**Mistake Tracking**: Incorrectly matched verbs are flagged and stored locally. The start screen surfaces a "Needs repetition" list so players can run a focused practice round on their weak verbs.

**Round Summary**: After every completed round, a summary modal lists all matched verbs and their definitions alongside the time taken, and lets players continue to the next round or return home.

**Grammar Note**: A built-in modal explains the three types of phrasal verbs — separable, inseparable, and intransitive — with examples.

**Word List**: 300+ curated phrasal verbs browsable via a searchable in-game table, stored entirely client-side with no backend required.

## Tech Stack

Vanilla HTML, CSS, and JavaScript. No build step or dependencies — open `index.html` directly in a browser.
